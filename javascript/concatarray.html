Concat Array dengan destructuring

Apa yang harus dilakukan apabila terjadi Conflict?

Hi teman-teman kali saya coba share tentan concat array di javascript dengan destructuring. Nah kebetulan saya perlu melakukan hal ini untuk kebutuhan pengambilan data berkala, ok langsung saja: 

                                
    let json1 = [{nama : 'rahmad'}, {nama: 'budi'}];
    let json2 = [{nama : 'yudha'}, {nama: 'ziqqi'}];

    let merge = [...json1, ...json2];
    console.log(merge);
Jadi nantinya jadinya akan seperti ini: 

                                
    [{nama: "rahmad"},{nama: "budi"},{nama: "yudha"},{nama: "ziqqi"}]
Nah sekian dulu sharing dari saya, jika kekurangan atau kritik kalian bisa beri masukan ke saya. Terimakasih dan semoga bermanfaat.
